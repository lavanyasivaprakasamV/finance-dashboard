# Finance Dashboard

## 🚀 Overview
A simple and interactive finance dashboard built using React and Vite. It allows users to track income, expenses, and view financial insights.

## 🛠 Tech Stack
- React.js
- JavaScript
- Recharts
- CSS
- Vite

## ✨ Features
- Summary cards (Balance, Income, Expense)
- Transaction list with search
- Add transactions (Admin role)
- Role-based UI (Viewer/Admin)
- Insights (highest spending category)
- Balance trend chart
- Responsive design
- LocalStorage persistence

## 📦 Installation
```bash
npm install
npm run dev
